Our weighted grade calculator shows your average and what to earn for the final grade you want. A timesaver if you don't know how to calculate grades!

[https://www.calculator.io/grade-calculator/](https://www.calculator.io/grade-calculator/)

== Usage ==

Add the Grade Calculator shortcode to your page, post or sidebar:

`[ci_grade_calculator]`

Add the following code to your template where you would like the Grade Calculator to appear:

`<?php display_ci_grade_calculator(); ?>`

== Installation ==

1. Upload the Grade Calculator /ci_grade_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Grade Calculator plugin through the "Plugins" menu in WordPress.
